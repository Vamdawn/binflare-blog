# Binflare Blog UI 重设计（Design）

- 日期：2026-03-14
- 状态：已评审通过（用户确认）
- 适用范围：`apps/web` 文章列表页（`/`）与文章详情页（`/posts/:slug`）

## 1. 背景与目标

在不改动路由与数据链路前提下，重做博客整体视觉与交互，目标风格为「理性高端」。
本次改版成功标准优先级：阅读完成率 > 点击深度 > 品牌识别。

## 2. 已确认设计决策

### 2.1 总体方向

- 视觉气质：理性高端（编辑感、克制、内容优先）
- 首页信息架构：纯文章流
- 详情页阅读辅助：强辅助（章节目录 + 阅读进度 + 回到顶部）
- 主题策略：仅浅色模式
- 列表密度：中密度

### 2.2 信息架构与页面骨架

#### 首页 `/`

- 顶部仅保留品牌区（`Binflare Blog`）+ 轻量导航入口
- 不设置营销型 Hero，首屏直接进入文章流
- 文章卡片字段：标题、日期、2 行摘要、可选阅读时长、`阅读全文`
- 响应式：桌面双列，移动端单列

#### 详情页 `/posts/:slug`

- 窄栏正文布局（控制行长 65-72 字符）
- 桌面右侧固定阅读辅助区：
  - 章节目录（高亮当前 section）
  - 阅读进度条
  - 回到顶部按钮
- 移动端：阅读辅助收敛为折叠入口

### 2.3 视觉系统（Design Tokens）

#### 颜色

- `--bg: #F5F7FA`
- `--surface: #FFFFFF`
- `--text: #0F172A`
- `--text-muted: #475569`
- `--border: #DCE3EB`
- `--accent: #1D4ED8`
- `--accent-strong: #1E40AF`
- `--focus-ring: #93C5FD`

#### 字体

- 标题：`Newsreader`
- 正文/UI：`Roboto`
- 等宽：`JetBrains Mono`

#### 字号与节奏

- 正文：`18px`，`line-height: 1.72`
- 元信息：`13-14px`
- 间距体系：`4/8/12/16/24/32/48`

#### 视觉细节原则

- 以边框层级和留白取代厚重阴影
- 强调色仅用于交互反馈
- 圆角控制在 `12-14px` 中小尺度

### 2.4 交互与阅读辅助

- 动效：150-220ms，且仅使用 `opacity/transform`
- 支持 `prefers-reduced-motion`，关闭位移动画和平滑滚动
- 首页卡片：轻微上浮 + 边框增强
- 链接交互：下划线/颜色过渡，避免夸张动效
- 章节目录：点击跳转、滚动联动高亮（IntersectionObserver）
- 阅读进度：按正文滚动比例更新
- 回顶按钮：达到阈值后出现
- 渐进增强：无 JS 时正文保持完整可读

## 3. 可访问性与性能底线

### 3.1 可访问性

- 正文与背景对比度满足 WCAG AA（>=4.5:1）
- 全部可交互元素键盘可达、焦点可见
- 目录与回顶具备明确 `aria-label`
- 不仅依赖颜色表达状态
- 遵循 `prefers-reduced-motion`

### 3.2 性能

- 字体使用 `font-display: swap`
- 媒体预留尺寸，避免 CLS
- 阅读进度计算节流，避免滚动卡顿
- 阅读辅助不阻塞正文渲染

## 4. 边界行为

- 保持现有路由：`/`、`/posts/:slug`、`/404`
- 保持数据入口：`getPosts()`、`getPostBySlug()`
- 无效 slug：跳转 `/404`
- 无二级标题：隐藏目录模块，保留进度/回顶
- 超长代码块：允许横向滚动，不挤压正文列

## 5. 验收标准

- 响应式断点（375/768/1024/1440）无横向滚动
- 详情页目录高亮、阅读进度、回顶按钮可用且无明显卡顿
- 现有路由与文章数据链路回归测试通过
- 阅读路径较当前版本更稳定，正文干扰项减少
